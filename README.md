# testing-solidity

Learn more by reading the [blog post](https://yos.io/2020/07/09/testing-smart-contracts/).

## Getting Started

```bash
git clone git@github.com:yosriady/testing-solidity.git
cd testing-solidity
yarn test
```

## Generating code coverage

```bash
yarn coverage
```
